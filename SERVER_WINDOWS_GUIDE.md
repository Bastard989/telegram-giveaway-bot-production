# Сервер бота: инструкция для Windows

## Нужно ли заходить на сервер каждый день

Нет. После настройки сервер работает круглосуточно, а владелец управляет розыгрышами только через Telegram-бота. Сервер автоматически запускает бот после перезагрузки и каждый день создаёт резервную копию базы.

На сервер заходят только для диагностики, обновления или восстановления. Нельзя одновременно запускать эту же копию бота на Windows: Telegram разрешает только один процесс с одним токеном.

## Первый доступ с нового Windows-компьютера

1. Откройте меню «Пуск», найдите `PowerShell` и запустите его.
2. Создайте личный SSH-ключ:

   ```powershell
   ssh-keygen -t ed25519 -C "giveaway-server"
   ```

3. На вопрос о пути нажмите Enter. Пароль ключа можно задать или дважды нажать Enter.
4. Покажите публичную часть ключа:

   ```powershell
   Get-Content "$env:USERPROFILE\.ssh\id_ed25519.pub"
   ```

5. Отправьте администратору только полученную строку, начинающуюся с `ssh-ed25519`. Файл `id_ed25519` без `.pub` никому не отправляйте.
6. После добавления ключа подключитесь:

   ```powershell
   ssh root@IP_СЕРВЕРА
   ```

7. При первом подключении напишите `yes` и нажмите Enter.

Если появляется `Permission denied (publickey)`, публичный ключ этого компьютера ещё не добавлен на сервер.

## Как смотреть файлы через Midnight Commander

После подключения выполните:

```bash
cd /opt/telegram-giveaway-bot
mc
```

Основные клавиши:

- стрелки — выбрать файл или папку;
- Enter — открыть папку;
- `Tab` — перейти между левой и правой панелью;
- `F3` — посмотреть файл без изменения;
- `F4` — редактировать файл;
- `F5` — скопировать;
- `F6` — переместить;
- `F7` — создать папку;
- `F8` — удалить;
- `F10` — выйти из `mc`.

На ноутбуке функциональные клавиши иногда нажимаются вместе с `Fn`.

Важные файлы и папки:

- `/opt/telegram-giveaway-bot/.env` — секретные настройки и токен; не пересылать и без необходимости не менять;
- `/opt/telegram-giveaway-bot/runtime/giveaway-bot.sqlite3` — база розыгрышей;
- `/opt/telegram-giveaway-bot/runtime/backups` — ежедневные резервные копии;
- `/opt/telegram-giveaway-bot/app.py` — точка запуска программы.

Не удаляйте `.env`, `.venv`, `runtime` и не редактируйте SQLite-базу вручную.

## Безопасные команды проверки

Состояние бота:

```bash
systemctl status telegram-giveaway-bot --no-pager
```

Последние сообщения журнала:

```bash
journalctl -u telegram-giveaway-bot -n 100 --no-pager
```

Перезапуск бота:

```bash
systemctl restart telegram-giveaway-bot
```

Проверка ежедневных backups:

```bash
systemctl list-timers telegram-giveaway-backup.timer
ls -lh /opt/telegram-giveaway-bot/runtime/backups
```

Выход с сервера:

```bash
exit
```

Не запускайте `python app.py` вручную: постоянным процессом управляет системная служба.
