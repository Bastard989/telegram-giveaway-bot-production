# Руководство использования

## 1. Запуск панели

На macOS откройте:

```text
START.command
```

На Windows откройте:

```text
START.bat
```

Панель запуска откроется в браузере сама.

## 2. Настройка

В панели заполните:

- токен бота из BotFather;
- Telegram ID владельца или username владельца;
- часовой пояс;
- фразу для участия через комментарии.

Telegram ID не обязателен, если указан username. Но ID надежнее, потому что username можно поменять.

## 3. Подготовка

Нажмите:

1. `Сохранить настройки`
2. `Установить нужные файлы`
3. `Подготовить базу данных`
4. `Запустить бота`

Для подготовки базы данных нужен Docker Desktop.

## 4. Создание розыгрыша

В Telegram напишите боту:

```text
/start
```

Нажмите `Создать розыгрыш` и заполните шаги:

1. Тип розыгрыша: по кнопке или по комментариям.
2. Название.
3. Текст поста.
4. Фото/видео или `нет`.
5. Дату окончания.
6. Количество призовых мест.
7. Количество победителей на каждое место.
8. Капчу, если нужна.

## 5. Как работают призовые места

Первое число - это количество мест.

Второе число - сколько людей будет на каждом месте.

Примеры:

- `1` место и `1` победитель на место = всего 1 победитель.
- `5` мест и `1` победитель на место = 5 победителей, по одному на места 1-5.
- `5` мест и `2` победителя на место = 10 победителей, по два человека на каждое место.
- `10` мест и `3` победителя на место = 30 победителей.

## 6. Условия подписки

Есть два типа условий:

- строгая проверка: бот реально проверяет подписку, но должен быть админом в канале;
- мягкая ссылка: бот показывает кнопку подписки, но не проверяет подписку.

Для личного канала, куда не хочется давать админку, используйте мягкую ссылку.

## 7. Ручной выбор победителя

Для активного розыгрыша нажмите:

```text
Выбрать победителя вручную
```

Отправьте username участника:

```text
@username
```

Этот пользователь станет одним из победителей первого места. Остальные места бот заполнит автоматически из других участников.

Важно: пользователь уже должен быть среди участников розыгрыша.

## 8. Завершение

Розыгрыш завершится автоматически по дате окончания или вручную кнопкой:

```text
Завершить сейчас
```

После завершения бот отправит результаты в канал публикации и владельцу бота.
